Problem of the Week (P01)

Show that there are no continuous functions $f, g$ and $h$ defined on $\mathbb{R}$ satisfying$$h(f(x) + g(y)) = xy \text{ for all } x, y \in \mathbb{R}.$$
Proof.

Fix $x$ and $y$ separately to show $f, g$, and $h$ must be continuous, strictly monotonic functions.

Since $h$ is strictly monotonic, it is injective, implying $f(x)+g(y)$ must be equal whenever $xy$ is equal.

Evaluating $xy = 1$ for $(1,1)$ and $(-1,-1)$ gives the relation $f(1) + g(1) = f(-1) + g(-1)$.

Evaluating $xy = -1$ for $(1,-1)$ and $(-1,1)$ gives the relation $f(1) + g(-1) = f(-1) + g(1)$.

Subtracting these equations yields $g(1) = g(-1)$, which contradicts the strict monotonicity required for $g$.

Theorem of the Week (T01)

Theorem (AM-GM for two positive reals). For all positive real numbers $a, b$, we have $$\frac{a+b}{2} \geq \sqrt{ab}$$ with equality if and only if $a=b$.
Proof. Since $(a-b)^2 \ge 0$ for all real $a,b$, expanding gives $$a^2 - 2ab + b^2 \ge 0 \Rightarrow (a+b)^2 \ge 4ab.$$ Because $a,b>0$, both sides are nonnegative, so taking square roots yields $$\frac{a+b}{2} \ge \sqrt{ab}.$$ Equality holds exactly when $(a-b)^2=0$, i.e., when $a=b$. $\blacksquare$
